# My dotfiles

This directory cotains the dotfiles for my xfce4/i3 Arch Linux system.

## Requirements

Ensure you have the following on your system!

### Git

$ sudo pacman -S git 

### Stow

$ sudo pacman -S stow 

## Installation

$ git clone https://github.com/cherrydextrous/dotfiles.git
$ cd dotfiles

$ stow .
